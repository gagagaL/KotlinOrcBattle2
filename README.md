# KotlinOrcBattle2

## What's
  
兵士を操作し、12の敵を倒す  

### グローバル変数  
  
兵士は体力(health)、素早さ(agility)、力(strength)の三つの状態を持つ  
体力が0になるとゲームオーバー、素早さは1ラウンドで兵士が何回攻撃できるか、力は攻撃の威力を示す  
  
### その他指示されているグローバル変数  
  
monsters...モンスターについての情報を保持する配列  
monster-builders...モンスターを作成する関数のリストを格納する  
monster-num...モンスターの総数、難易度調整はここで行う
